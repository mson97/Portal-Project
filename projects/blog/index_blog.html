🔖 Post 1

HTML stands for Hyper Text Markup Language. It describes the structure of a Webpage. HTML consists of a series of elements. The HTML elements tell the browser, like Chrome or Firefox, how to display the content. HTML elements are represented by tags. Tags will label pieces of content that correlate with their purpose such as "heading", "paragraph", and "footer." 🔍Read More

Sir Tim Berners-Lee created HTML 1991. However, HTML was not officially released later in 1993. The next version of HTML was dubbed HTML 2.0 and was released in 1995. The next major version of HTML was published in 1999 as HTML 4.01. Sir Berners Lee is an English computer scientist and engineer. He is also known as the invented of the World Wide Web. He is currently 64 years old and is the direct of W3C, the web standards organization. HTML goes through many changes constantly and evolved with each update. 🔍Read More
