Intro to HTML

HTML stands for Hyper Text Markup Language. It describes the structure of a Webpage.

HTML consists of a series of elements. The HTML elements tell the browser, like Chrome or Firefox, how to display the content.

HTML elements are represented by tags. Tags will label pieces of content that correlate with their purpose such as "heading", "paragraph", and "footer."

Browsers won't display the tag themselves, but the tags will indicate to the browser to format the content of the page.

HTML documents always begin with a initial starting code that tells the browser that it's HTML document type. It will consists of the head and the body.

HTML tags normally come in pairs like where the first tag is the start tag, and the second tag is the end tag. The end tag contains a forward slash inserted before the tag name. The tags are enclosed in angle brackets. The tags enclose the content that they represent.

A web browser reads HTML documents and displays them. The browser determines how to display the document. For example, the web browser knows to make headings larger in font size than a paragraph. It is often helpful to visualize the structure of your webpage and associate each block with a corresponding tag.
